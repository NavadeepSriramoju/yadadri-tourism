## **OPTION 5: FIREBASE HOSTING** 🔥 (GOOGLE POWER)

```

```

### How to deploy:
```bash
npm install -g firebase-tools
firebase login
firebase init hosting

# In firebase.json, set:
# "public": "frontend/build"

cd frontend && npm run build
cd ..
firebase deploy
```
